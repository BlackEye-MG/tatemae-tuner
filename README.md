# Tatemae Tuner

This is a browser extension built with React, TypeScript, and Vite.

## Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/BlackEye-MG/tatemae-tuner.git
    cd tatemae-tuner
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Build the extension:**
    ```bash
    npm run build
    ```
    This will create a `dist` directory with the extension files.

## Usage

1.  Open your browser's extension page:
    *   **Chrome:** `chrome://extensions`
    *   **Firefox:** `about:addons`
    *   **Edge:** `edge://extensions`

2.  Enable **"Developer mode"** (usually a toggle switch).

3.  Click **"Load unpacked"** and select the `dist` directory from this project.

The Tatemae Tuner extension should now be installed and active in your browser.

## Development

To run the project in development mode with hot-reloading:

```bash
npm run dev
```
